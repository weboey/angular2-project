Inputs

disabled

是否禁用. 数据类型 boolean, 默认false;

Type: boolean

Default value: false

marks

marks 标签 使用格式为 [Object] 其中 Object 必须包含value 及label 可以有style 属性 例如: marks = [{value: 20, label: '20 ℃'},

Type: SliderMark[]

max

输入范围的可选最大值.
min

可选范围的最小值
step

每次变化的最小值, 最小支持小数点后两位.
value

slider的当前值, 类型 number | ArrayCollection 支持多触点.

Type: number | ArrayCollection

vertical

垂直滑动条 默认 false

Type: boolean

Default value: false

Outputs

change
valueChange

使 value 支持双向绑定
